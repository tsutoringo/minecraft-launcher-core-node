lookupByName

  • lookupByName(name: string, option?: { api?: ProfileServiceAPI; timestamp?: undefined | number }): Promise<GameProfile>
    • Defined in packages/user/service.ts:150

    Look up the GameProfile by username in game. This will return the UUID of the name at the timestamp provided. ?at=0 can be used to get the UUID of the original user of that username, but, it only works if the name was changed at least once, or if the account is legacy.

    The timestamp is a UNIX timestamp (without milliseconds) When the at parameter is not sent, the current time is used

    throws

    ProfileLookupException

    Parameters

    • name: string

      The username in game.

    • Default value option: { api?: ProfileServiceAPI; timestamp?: undefined | number } = {}

      • Optional timestamp?: undefined | number

    Returns Promise<GameProfile>

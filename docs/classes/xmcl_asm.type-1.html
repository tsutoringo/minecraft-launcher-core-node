The offset of the internal name of this Java type in {@link #buf buf} or, for primitive types, the size, descriptor and getOpcode offsets for this type (byte 0 contains the size, byte 1 the descriptor, byte 2 the offset for IALOAD or IASTORE, byte 3 the offset for all other instructions).

Static getArgumentsAndReturnSizes

  • getArgumentsAndReturnSizes(desc: string): number
    • Defined in packages/asm/libs/Type.ts:259

    Computes the size of the arguments and of the return value of a method.

    Parameters

    • desc: string

      the descriptor of a method.

    Returns number

    the size of the arguments of the method (plus one for the implicit this argument), argSize, and the size of its return value, retSize, packed into a single int i = (argSize << 2) | retSize (argSize is therefore equal to i >> 2, and retSize to i & 0x03).
